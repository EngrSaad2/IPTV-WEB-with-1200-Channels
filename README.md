## Features

- **Live TV Streaming:** Access to over 1200 channels streaming live via IPTV feeds.
- **Categorized Channel Access:** Filter channels by category, including Bangladeshi, Movies, Sports, Global, and Islamic.
- **Customized Channel Sourcing:** Integration of custom movie and sports feeds including Star Movies, T Sports HD, beIN Sports, and many more.
- **Movie Database:**
  - Trending movies list.
  - New releases list.
  - Top-rated movies list.
  - Search movies by title.
  - View details by movie.
  - Categorization by genre.
- **Favorites:** Add favorite TV channels and movies for quick access.
- **User Profile Management:** Customize user preferences and options.
- **Fast Caching:** Integrated Cache mechanism to fetch IPTV channel data quickly and efficiently.
- **Modern Responsive UI:** Optimized layout tailored for mobile devices, tablets, and desktop browsers.

## Technical Stack

- **Framework:** Laravel 11 / PHP 8.2+
- **Styling:** Custom CSS
- **Proxy Caching:** Integrated proxy controller caching raw IPTV feeds for maximum response time.
- **CI/CD:** Automated GitHub Actions SSH/rsync pipeline for zero-downtime deployment.
